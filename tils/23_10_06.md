### **Fallback 데이터**

- **정의**: 원본 데이터가 사용 불가능하거나 접근할 수 없을 때 대신 사용되는 예비 데이터를 "fallback 데이터"라고 함
- **목적**: 데이터의 일시적 또는 영구적 손실, 네트워크 장애 등 다양한 상황에서 시스템의 가용성과 안정성을 유지하기 위함
- **사용 예**:
    1. 웹사이트에서 이미지 로딩 실패 시 표시되는 기본 이미지
    2. 네트워크 문제로 API 호출이 실패한 경우 로컬 캐시에서 불러오는 데이터
    3. 라이브 스트리밍 중 네트워크 지연이 발생하면, 서버는 미리 저장된 비디오 클립을 재생하여 사용자 경험을 해치지 않게 함
